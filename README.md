## Introduction and Overview

This QR code generator would feature:
- Correctly generate QR codes with proper structure (finder, timing, format patterns)
- Masking to improve the scan reliability
- Simple web UI for user input and QR display built with flask
- Customisable styling for QRs (colour, size)
- Byte-mode, Alphanumeric and numeric encoding
- 4 levels of error correction (L, M, Q, H)
